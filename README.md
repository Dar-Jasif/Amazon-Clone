# Amazon Clone

A simple Amazon-inspired e-commerce UI built using **React** and **Vite**.  
It includes product listing, product details, cart management, and basic checkout flow.

## Tech Stack

- React (Vite)
- JavaScript (ES6+)
- HTML5, CSS3
- ESLint for linting

## Features

- Home page with product listings
- Product detail page
- Add to cart / remove from cart
- Cart page with summary
- Basic checkout page layout
- Responsive layout for different screen sizes
